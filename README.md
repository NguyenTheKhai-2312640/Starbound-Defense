# Starbound-Defense
Nhóm 14 - Môn Phát triển ứng dụng Game - Unity 2D - Game thể loại thủ thành chủ đề vũ trụ


## Mô tả:

Game 2D dạng thủ thành lấy bối cảnh bạn vào vai một phi hành gia thuộc đội khai thác tài nguyên liên hành tinh. Trong lúc đó, tín hiệu từ con tàu đã thu hút lũ sinh vật kỳ lạ (Aliens). Nhiệm vụ của bạn là bảo vệ con tàu không để bị phá hủy cho đến khi thu thập đủ nhiên liệu hoặc thu thập xong mẫu vật.
